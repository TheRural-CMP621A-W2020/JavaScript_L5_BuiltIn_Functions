Random Number Generation


This is very different from random generation in Python. Python you can import a library that is full of different type of random generators. Many programming languages do not have a large number of generators like Python. Whenever you want to generate a number in JavaScript you need to have an understanding of how Math.random() works and some basic math to create the type of number you want for your program. The generators in JavaScript does seem to be pretty random. Some programming languages the numbers aren't very random at all.

Math.random() will generate a decimal 0<=x<1. Try for yourself.


But if you want to create a random generator that is between 1 to 10 for example but your generator only does decimals then you are going to have to apply some math. To calculate a random number between 1 to 10 use the following code.

Math.floor((Math.random() * 10) + 1);
