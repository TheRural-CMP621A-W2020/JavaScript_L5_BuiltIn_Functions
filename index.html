Student Exercise(s)

  1. Build a random generator that generates integers between 1 and 100.
  2. Build a random generator that generates integers between 20 and 101.
  3. Build a random generator that generates numbers between 1 and 100 but they must have one decimal. Examples are 58.7, 94.6, 9.5, 22.4.
  4. Build a program that will allow a user to input a word. The program will display the second and second last letter of the word.
